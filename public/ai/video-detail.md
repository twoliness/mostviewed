# YouTube Video Performance Detail

Per-video performance page showing rank history, daily view trends, peak rank, and chart timeline for any tracked YouTube video.

Each video detail page includes:
- Current chart rank and peak rank across global and category charts
- Days on chart and total view count
- Rank timeline chart showing rank position over time across all charts the video appeared in
- Daily view delta history (how many views the video gained each day)
- Creator profile with subscriber count and other videos from the same channel

This is a live, continuously updated page on Most Viewed Today (https://mostviewed.today). Numbers on this static summary are not live — follow the canonical URL for current rankings, view counts, and timestamps.
